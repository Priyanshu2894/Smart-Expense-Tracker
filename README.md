# 🚀 SmartExpense Tracker
A full-stack financial management application that helps users track spending, visualize habits, and get AI-powered savings advice.

## ✨ Key Features
Real-time Tracking: Log expenses and income with instant balance updates.

AI Savings Advice: Integrated with Gemini 1.0 Pro to analyze spending patterns and suggest budget improvements.

PDF Reports: One-click generation of professional financial statements using jsPDF.

Data Visualization: Interactive charts to see spending "By Category."

Dark Mode: Sleek, modern interface designed for accessibility.

## 🛠️ Tech Stack
Frontend: React.js, Tailwind CSS, Vite

Backend: Node.js, Express.js

Database: MongoDB

State Management: Redux Toolkit

AI Support: Google Gemini API

⚙️ Installation & Setup
Clone the project:
Bash
git clone https://github.com/YourUsername/Smart-Expense-Tracker.git

Setup Backend:
Bash
cd server
npm install
# Create a .env file and add your MONGO_URI and GEMINI_API_KEY
npm start

Setup Frontend:
Bash
cd client
npm install
npm run dev
